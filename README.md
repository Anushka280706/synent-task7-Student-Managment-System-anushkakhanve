# Student Management System

A beginner-friendly Student Management System built using Python that allows users to manage student records through a Command Line Interface (CLI). The project supports adding, viewing, updating, and deleting student information with permanent storage using JSON files.

# Features

* Add student records
* View student details
* Update existing records
* Delete student records
* JSON-based file storage

# Technologies Used

* Python
* JSON

# Run the Project

```bash
python student_management.py
```

#Files

* `student_management.py` – Main program
* `students.json` – Stores student data
